 # Whac-a-mole-Game
A simple browser-based Whac-a-Mole game built with vanilla JavaScript, HTML, and CSS — no frameworks or libraries required.

## Features

- 3x3 grid with randomized mole spawning
- Live score tracking
- Countdown timer with game-over detection
- Restart button to reset and play again
- CSS keyframe animations for a pop-up, 3D mole effect

## How to play

Click (or tap) on the mole as it appears in one of the nine holes before it disappears. Each successful hit increases your score. The game ends when the timer reaches zero — click Restart to play again.

## Built with

- HTML5
- CSS3 (Grid layout, keyframe animations)
- JavaScript (DOM manipulation, `setInterval`/`setTimeout`)

## Running locally

Clone the repo and open `index.html` in your browser — no build step or dependencies needed.
